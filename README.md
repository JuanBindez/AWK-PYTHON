## Você Precisa Ter o Git Instalado:

`sudo apt install git`

## Download:

`git clone https://github.com/JuanBindez/AWK-PYTHON/`

## Dentro Do Diretório Onde Esta o Script awk.py Digite o Comando:

`python3 awk.py`
